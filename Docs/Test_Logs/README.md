# Test Logs

Store validation notes and command transcripts for FastMD here.

Recommended file naming:

- `manual-smoke-YYYYMMDD-HHMMSS.md` for interactive runs
- `validation-YYYYMMDD-HHMMSS.md` for build and test verification notes

Each log should record:

- machine date and time
- macOS version if relevant
- commands executed
- pass or fail outcome
- any follow-up investigation needed
